# 🎫 TicketHub Backend - Flask Application

TicketHub is a simple event booking backend built using Flask. It provides APIs for browsing events, searching, filtering, and booking tickets, along with a minimal frontend interface.

---

## 🚀 Features

- View all available events
- Search events (secure & unsafe versions for testing)
- Filter events by category, price, and date
- Book tickets
- View user bookings
- Event statistics
- Basic frontend UI (served via Flask)

---

## 🛠️ Tech Stack

- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **Data Storage:** In-memory (no database)
